한가족공동체 "천인방 "은 1천명으로 구성된 미래형 차세대 단체로 인맥적 경제 활동과 내부 수익등 경제적 이익을 공유하는 가족적 개념이 적용된 "공유경제 공동체"입니다.
살아가면서 경제적 어려움을 피할수 있도록 예비하고 싶다면 천인방 가족이 되십시오. 권속이 되시면 긴급한 모든 경제적 문제를 인맥과 재정으로 공동체에서 적극 지원합니다.
